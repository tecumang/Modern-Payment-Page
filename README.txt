🚀 Modern Payment Page Bundle (Success + Failed)

Take your payment experience to the next level with this clean, responsive, and easy-to-use HTML template bundle. Whether you're a developer, student, or freelancer, these pages are plug-and-play and built to impress.

📦 What's Included

✅ Success Page – A beautiful confirmation page with animated confetti effect.
❌ Failed Page – A clean error-style UI for failed transactions.
🌐 Fully responsive layout using MDB Bootstrap framework.
🧩 Well-commented and organized code.
🎯 Ideal for integration in Laravel, PHP, static HTML, or any custom backend.

🛠️ Tech Stack

- HTML5  
- CSS3  
- MDB Bootstrap 5  
- Custom animations (confetti)

📋 How to Use

1. Download and unzip the folder.
2. Open `index.html` in your browser to preview.
3. Integrate into your project or backend by redirecting users to these pages after payment.
4. Customize text, colors, and logos as needed.
